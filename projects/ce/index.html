    Course Evaluations is a Web based tool that caters to student facing survey, administrator facing reporting suite for higher education campuses.

    entity-framework azure-webapps azure-devops azure-functions dotnet-core

    Senior Software Developer

    :key: Key Contributions#

    • Team was responsible for architecting and delivering high volume, business critical mass mailing solutions using azure’s serverless functions. The initiative improved Service Level Agreement and end-to-end visibility for all stakeholders
    • I was involved in revamping legacy reporting microservice using Azure Data Factory and Azure Datalake Analytics. The solution opened new avenues of seamless integration with third party systems.

    Day to Day Contributions#

    • Involved in design and implementation of serverless mass mailing infrastructure using Azure Functions

    • Designed and architected data pipelines using Azure Data Factory and Azure Datalake Analytics

    • Contributed to migrating/restructuring multiple services as part of on-prem to azure migration

    • Researched reporting solutions for the team to move away from on premise SSIS (SQL Server Integration Services).

    • Gained in-depth experience with working on Azure App Services. Worked extensively on setting up improving / monitoring of student facing web application

    • Led implementation of data integration system with third party reporting systems.

    • Worked as sole full stack developer for MongoDB based node application

    • Hands on experience with building RESTful API in dotnet core, C# using multiple ORMs including Dapper, NHibernate and Entity Framework

    • Worked across multiple teams to coordinate troubleshooting production issues.

    • Helped onboard new members to the team.

    What problems did I solve ?#

    • Led efforts to move to a serverless solution for mailing with a strong focus on reliability. The new mailing system provided better transparancy around client notification in general and help reduce support-dev interactions for that topic
    • Led efforts to make third party event driven to account for intermittent failures between systems and improving tracking outliers
